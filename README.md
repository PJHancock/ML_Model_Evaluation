# **ML Model Evaluation**

This repository is a public resource to help people learn how to evaluate machine learning models.
It focuses on making evaluation easier to understand and apply, whether you are a student, researcher, or engineer.

## **What this repo is for**

* Learn basic model evaluation techniques
* Understand common metrics (accuracy, precision, recall, etc.)
* Compare different machine learning models
* Build intuition for how models perform in practice

## **Getting started**

To get started, simply clone the repo and explore the different labs:

```
git clone https://github.com/PJHancock/ML_Model_Evaluation
cd ML_Model_Evaluation
```

## **Goal**

Machine learning is not just about training models—it’s also about understanding how well they work.  This repo is meant to make that process clearer and easier to learn from.  This repository will help you better understand and compare machine learning models in a simple and practical way.
